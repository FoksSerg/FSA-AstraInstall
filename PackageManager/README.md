# PackageManager

Модуль управления пакетами для FSA-AstraInstall.

**Версия:** V3.1.159 (2025.12.03)  
**Компания:** ООО "НПА Вира-Реалтайм"  
**Разработчик:** @FoksSegr & AI Assistant (@LLM)

## 📋 Описание

PackageManager - это GUI инструмент для управления пакетами и создания архивов компонентов проекта FSA-AstraInstall. Модуль предоставляет:

- ✅ Создание и управление компонентами
- ✅ Формирование архивов компонентов
- ✅ Работа с файловой структурой компонентов
- ✅ Создание сборок (bundles) компонентов
- ✅ Автоматическое создание шаблонов Wine приложений
- ✅ Параллельная архивация с прогрессом

## 🚀 Быстрый старт

### Запуск GUI

```bash
# Способ 1: Через Python модуль
cd /Volumes/FSA-PRJ/Project/FSA-AstraInstall
python3 -m PackageManager.run_gui

# Способ 2: Прямой запуск скрипта
python3 PackageManager/run_gui.py

# Способ 3: Прямой запуск GUI модуля
python3 PackageManager/gui.py
```

### Режим Archive Worker

Модуль может работать в режиме воркера для параллельной архивации:

```bash
python3 PackageManager/gui.py --archive-worker <config_file> <progress_file>
```

## 📁 Структура модуля

### Основные компоненты

1. **Вспомогательные функции** (строки 30-607):
   - `load_components_from_archives()` - загрузка конфигураций из архивов
   - `extract_config_from_archive()` - извлечение конфигурации
   - `auto_create_wine_template_packages()` - создание шаблонов Wine

2. **Archive Worker** (строки 608-869):
   - `archive_worker_create_archive()` - создание архивов
   - `archive_worker_update_progress()` - обновление прогресса
   - `archive_worker_main()` - главная функция воркера

3. **GUI класс** (строки 871-6280):
   - `PackageManagerGUI` - основной класс GUI с 7 вкладками

## 🔧 Зависимости

- **astra_automation.py** - для импорта `COMPONENTS_CONFIG` (fallback)
- **AstraPack/** - директория с архивами компонентов
- **tkinter** - для GUI интерфейса

## 📝 Использование

### Создание компонента

1. Откройте вкладку "Создание компонента"
2. Заполните поля конфигурации
3. Добавьте файлы/директории компонента
4. Нажмите "Создать архив"

### Создание сборки

1. Откройте вкладку "Создание сборки"
2. Выберите компоненты для сборки
3. Разрешите зависимости
4. Нажмите "Создать архив сборки"

### Архивация структуры

1. Откройте вкладку "Архивация"
2. Выберите группы для архивации
3. Настройте исключения/дополнения
4. Нажмите "Архивировать выбранное" или "Архивировать все"
