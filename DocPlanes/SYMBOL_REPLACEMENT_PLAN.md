## Проблема

На Astra Linux 1.7 эмодзи (Unicode символы из диапазона U+1F300 и выше) часто не отображаются корректно или отображаются как квадратики. Однако многие спецсимволы из более ранних диапазонов Unicode работают нормально.

## Результаты тестирования

**Дата тестирования:** 2025-12-11 00:23:05  
**Платформа:** Astra Linux 1.7/1.8  
**Python:** 3.11.2  
**Кодировка:** utf-8

**Статистика:**
- Всего протестировано: **211 символов**
- Работают: **208 символов** (98.6%)
- Не работают: **3 символа** (1.4%)

### Не работающие символы

Не работают только варианты эмодзи с модификатором `U+FE0F` (emoji variant selector):

| Символ | Unicode | Название | Статус |
|--------|---------|----------|--------|
| ⚠️ | U+26A0 U+FE0F | WARNING SIGN (emoji variant) | ✗ Не работает |
| ❄️ | U+2744 U+FE0F | SNOWFLAKE (emoji variant) | ✗ Не работает |
| ⚙️ | U+2699 U+FE0F | GEAR (emoji variant) | ✗ Не работает |

**Важно:** Базовые версии этих символов **БЕЗ** модификатора работают:
- ⚠ (U+26A0) — работает
- ❄ (U+2744) — работает
- ⚙ (U+2699) — работает

### Вывод

**Эмодзи можно использовать в коде!** Практически все эмодзи работают на Astra Linux 1.7/1.8. Нужно только избегать вариантов с модификатором `U+FE0F`.

## Рекомендуемые замены для UI

### В кнопках и заголовках:

1. **Папка/Директория:**
   - `📁 Управление директориями` (эмодзи работает!)
   - Альтернатива: `[DIR] Управление директориями` (текстовая)

2. **Файл:**
   - `📄` (эмодзи работает!)
   - Альтернатива: `[FILE]` (текстовая)

3. **Снимок/Камера:**
   - `📸 Создать снимок` (эмодзи работает!)
   - Альтернатива: `[IMG] Создать снимок` (текстовая)

4. **Настройки/Конфигурация:**
   - `🔧` или `⚙` (оба работают!)
   - Альтернатива: `[CFG]` (текстовая)

### В статусах:

- `✓` (галочка) - работает, использовать для "OK"
- `✗` (крестик) - работает, использовать для "Ошибка"
- `●` (чёрный круг) - работает, использовать для "Включено"
- `○` (белый круг) - работает, использовать для "Выключено"

### В списках и навигации:

- `▶` (треугольник вправо) - работает, использовать для "Развернуть"
- `◀` (треугольник влево) - работает, использовать для "Свернуть"
- `▼` (треугольник вниз) - работает, использовать для выпадающих списков
- `▲` (треугольник вверх) - работает, использовать для выпадающих списков
