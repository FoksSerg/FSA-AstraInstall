## 📖 Введение

**FSA-AstraInstall** — это графическое приложение для автоматизации управления Linux-системами. Программа позволяет обновлять операционную систему, устанавливать и удалять программы, управлять репозиториями и пакетами через удобный интерфейс.

Это руководство поможет вам быстро освоить основные функции программы.

---

## 🚀 Первый запуск

### Запуск программы

1. **Простой запуск:**
   ```bash
   ./FSA-AstraInstall
   ```
   
   ✅ **Автоматический перезапуск:** Программа автоматически определит, что требуются права администратора, и перезапустится с sudo. Вам не нужно запускать её вручную с sudo.

2. **После запуска** откроется главное окно с несколькими вкладками.

---

## 📑 Обзор вкладок

Программа состоит из следующих основных вкладок:

1. **Обновление ОС** — автоматическое обновление системы
2. **Установка Программ** — установка Wine и Astra.IDE
3. **Терминал** — встроенный системный терминал
4. **Мониторинг** — отслеживание запущенных процессов
5. **Пакеты** — управление установленными пакетами
6. **Репозитории** — управление APT-репозиториями
7. **Файловая система** — мониторинг изменений файлов
8. **О системе** — информация о системе

---

## 1️⃣ Вкладка "Обновление ОС"

### Основные функции

Эта вкладка предназначена для автоматического обновления операционной системы.

**Как использовать:**

1. **Режим тестирования (dry-run):**
   - Установите галочку "Режим тестирования" для проверки без реальных изменений
   - В этом режиме программа покажет, что будет сделано, но не выполнит изменения

2. **Запуск обновления:**
   - Нажмите кнопку **"Запустить"** для начала процесса обновления
   - Прогресс будет отображаться в логе выполнения и на прогресс-барах

3. **Остановка процесса:**
   - Нажмите кнопку **"Остановить"** для прерывания текущего процесса

### Элементы интерфейса

- **Лог выполнения** — показывает подробную информацию о выполняемых операциях
- **Детальный прогресс** — отображает прогресс по этапам:
  - **Загрузка** — загрузка пакетов из репозиториев
  - **Распаковка** — распаковка загруженных пакетов
  - **Настройка** — настройка установленных пакетов

---

## 2️⃣ Вкладка "Установка Программ"

### Основные функции

Эта вкладка позволяет устанавливать и удалять Wine и Astra.IDE.

**Как использовать:**

1. **Проверка компонентов:**
   - Нажмите кнопку **"Проверить компоненты"**
   - В таблице отобразятся все доступные компоненты и их статус (установлен/не установлен)

2. **Выбор компонентов:**
   - Отметьте галочками нужные компоненты в таблице
   - Можно выбрать несколько компонентов одновременно

3. **Установка:**
   - Нажмите **"Установить выбранные"** для установки отмеченных компонентов
   - Процесс установки будет отображаться в логе

4. **Удаление:**
   - Нажмите **"Удалить выбранные"** для удаления отмеченных компонентов

5. **Создание образа:**
   - Кнопка **"Создать Образ"** позволяет создать резервную копию Wine-префикса

### Элементы интерфейса

- **Таблица компонентов** — список всех доступных компонентов с их статусом
- **Колонки таблицы:**
  - ☐ — выбор компонента
  - Компонент — название компонента
  - Статус — текущее состояние (установлен/не установлен)
  - Путь/Детали — путь установки или дополнительная информация

---

## 3️⃣ Вкладка "Терминал"

### Основные функции

Встроенный системный терминал для выполнения команд.

**Как использовать:**

- Введите команду в поле ввода и нажмите Enter
- Результат выполнения команды отобразится в области вывода
- Используйте кнопку **"Очистить"** для очистки вывода

---

## 4️⃣ Вкладка "Мониторинг"

### Основные функции

Отслеживание запущенных процессов в системе.

**Как использовать:**

1. Таблица автоматически обновляется и показывает:
   - Имя процесса
   - PID (идентификатор процесса)
   - Использование CPU
   - Использование памяти
   - Время работы

2. **Обновление:**
   - Таблица обновляется автоматически каждые 0.5 секунды при открытой вкладке

3. **Сортировка:**
   - Кликните на заголовок колонки для сортировки по этому параметру

---

## 5️⃣ Вкладка "Пакеты"

### Основные функции

Управление установленными пакетами в системе.

**Как использовать:**

1. Таблица показывает все установленные пакеты с информацией:
   - Название пакета
   - Версия
   - Размер
   - Описание

2. **Поиск:**
   - Используйте поле поиска для быстрого поиска пакетов по названию

3. **Обновление:**
   - Таблица обновляется автоматически при открытии вкладки

---

## 6️⃣ Вкладка "Репозитории"

### Основные функции

Управление APT-репозиториями системы.

**Как использовать:**

1. **Загрузка репозиториев:**
   - Нажмите **"Загрузить репозитории"** для отображения всех репозиториев
   - Репозитории автоматически загружаются при открытии вкладки

2. **Проверка доступности:**
   - Нажмите **"Проверить доступность"** для проверки всех репозиториев по сети
   - Недоступные репозитории будут помечены

3. **Автоматическая проверка и обновление:**
   - Нажмите **"Автоматическая проверка и обновление"** для автоматической проверки и активации/деактивации репозиториев

4. **Обновление списков:**
   - Нажмите **"Обновить списки (apt update)"** для обновления локальной базы пакетов

5. **Управление репозиториями:**
   - **Правый клик** на репозитории в таблице открывает контекстное меню:
     - Просмотр деталей
     - Включить/Отключить репозиторий
     - Копировать URI
   - **Двойной клик** на репозитории открывает окно редактирования

6. **Редактирование репозитория:**
   - В окне редактирования можно изменить:
     - Статус (активен/неактивен)
     - Тип репозитория
     - URI
     - Дистрибутив
     - Компоненты
   - Нажмите **"Сохранить"** для применения изменений

### Элементы интерфейса

- **Таблица репозиториев** — список всех репозиториев с колонками:
  - Статус — активен/неактивен
  - Тип — тип репозитория (deb, deb-src и т.д.)
  - URI — адрес репозитория
  - Дистрибутив — версия дистрибутива
  - Компоненты — компоненты репозитория

- **Статистика** — показывает:
  - Всего репозиториев
  - Активных
  - Отключенных
  - Файл источников
  - Статус операций

## 💡 Полезные советы

### Общие рекомендации

1. **Запуск программы:**
   - Просто запустите программу: `./FSA-AstraInstall`
   - Программа автоматически перезапустится с sudo, если требуются права администратора
   - Вам не нужно запускать её вручную с sudo

2. **Режим тестирования:**
   - Перед выполнением важных операций используйте режим тестирования (dry-run)
   - Это позволит увидеть, что будет сделано, без реальных изменений

3. **Логи:**
   - Всегда проверяйте лог выполнения перед закрытием программы
   - Логи содержат подробную информацию о выполненных операциях

4. **Контекстное меню:**
   - Используйте правый клик мыши для доступа к дополнительным функциям
   - В разных вкладках доступны разные функции контекстного меню

5. **Подсказки:**
   - Наведите курсор на кнопки для отображения подсказок
   - Подсказки содержат краткое описание функции кнопки

## ⚠️ Важные предупреждения

1. **Резервное копирование:**
   - Перед выполнением важных операций рекомендуется создать резервную копию системы

2. **Проверка репозиториев:**
   - Перед обновлением системы проверьте доступность репозиториев
   - Недоступные репозитории могут вызвать ошибки при обновлении

3. **Удаление компонентов:**
   - Будьте осторожны при удалении компонентов
   - Убедитесь, что удаляемые компоненты не используются другими программами

## 🆘 Решение проблем

### Проблема: Программа не запускается

**Решение:**
- Запустите программу: `./FSA-AstraInstall` (программа автоматически перезапустится с sudo при необходимости)
- Проверьте, что файл имеет права на выполнение: `chmod +x FSA-AstraInstall`
- Если программа не может получить права sudo, убедитесь, что у вас есть права на выполнение команд с sudo

### Проблема: Ошибки при обновлении системы

**Решение:**
1. Перейдите на вкладку "Репозитории"
2. Нажмите "Проверить доступность"
3. Отключите недоступные репозитории
4. Попробуйте обновить систему снова

### Проблема: Компоненты не устанавливаются

**Решение:**
1. Проверьте доступность репозиториев
2. Обновите списки пакетов (apt update)
3. Проверьте логи выполнения на наличие ошибок
