# Хронология развития проекта FSA-AstraInstall

**Компания:** ООО "НПА Вира-Реалтайм"  
**Проект:** FSA-AstraInstall Automation  
**Текущая версия:** V4.8.380 (2026-04-09)

---

Документ отражает развитие проекта по версиям: от ранних автоматизаций и первых рабочих сборок до полноценного установщика с графическим интерфейсом, автономными пакетами, офлайн-репозиториями, монолитным архивом и развитой логикой подготовки системы.

Документ построен в обратной хронологии: вверху находятся последние изменения, внизу - начало проекта.

---

## 2025-11 (Расширение установочной платформы)

Проект вышел за рамки базовой автоматизации и начал превращаться в расширяемую установочную платформу с управлением компонентами, архивами, логированием и сборкой бинарных поставок.

- Добавлен графический инструмент для управления пакетами с возможностью установки, удаления и обновления компонентов.
- Расширена система управления компонентами Wine и добавлена поддержка Winetricks.
- Добавлен компонент ярлыка приложения и доработана работа с ярлыками для пользовательских сценариев.
- Улучшена логика обновления и введены приоритеты источников для более предсказуемой установки пакетов.
- Добавлена система сборки бинарных файлов для Linux через Docker с поддержкой платформ Astra 1.7 и 1.8.
- Улучшена работа установщика Wine-пакетов и снижено количество ошибок при удалении и повторной установке.
- Проведен рефакторинг системы управления прогрессом с переходом к более централизованной модели.
- Доработана распаковка архивов и улучшена обратная связь пользователю во время подготовки пакетов.
- Улучшена обработка логов, убраны избыточные отладочные сообщения и сокращено дублирование записей.
