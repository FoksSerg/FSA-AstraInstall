## 📋 Описание

Эти файлы были исходниками для сборки объединённого файла `FSA-AstraInstall.py`.

**Текущая ситуация:** Разработка ведётся напрямую с `FSA-AstraInstall.py`. Эти файлы больше не используются.

## 📁 Содержимое

- `astra_automation.py` - основной модуль с GUI (26,855 строк)
- `astra_update.sh` - скрипт обновления из сетевых источников (Linux)
- `astra_update.py` - скрипт обновления для macOS (292 строки)
- `astra_install.sh` - скрипт установки зависимостей и запуска (995 строк)
- `build_unified.py` - дубликат скрипта сборки из корня проекта

## 🔄 Исторический процесс сборки

Эти файлы объединялись скриптом `Build/build_unified.py` в единый файл `FSA-AstraInstall.py`:

1. `astra_automation.py` - основной модуль (встраивался полностью)
2. `astra_update.sh` - скрипт обновления (встраивался как строковая константа)
3. `astra_install.sh` - скрипт установки (встраивался как строковая константа)
4. `Build/self_updater.py` - модуль самообновления (встраивался как класс)
