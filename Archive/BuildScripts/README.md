## 📋 Описание

Эти скрипты использовались для создания объединённого файла `FSA-AstraInstall.py` из исходных файлов проекта.

**Текущая ситуация:** Разработка ведётся напрямую с `FSA-AstraInstall.py`. Эти скрипты больше не используются.

## 📁 Содержимое

- `build_unified.py` - скрипт объединения файлов проекта в единый Python файл
- `generate_unified.py` - обёртка для build_unified.py (вызывает build_unified.py через subprocess)
- `self_updater.py` - модуль самообновления (уже встроен в FSA-AstraInstall.py)
- `README_original.md` - оригинальная документация скриптов сборки

## 🔄 Процесс сборки (исторический)

1. **Генерация объединенного файла:**
   ```bash
   python3 Build/generate_unified.py
   ```
   Результат: `FSA-AstraInstall.py` в корне проекта

2. **Сборка бинарника (через DockerManager):**
   ```bash
   python3 -m DockerManager.cli --project FSA-AstraInstall --platform astra-1.8
   ```
   Результат: `FSA-AstraInstall` (бинарный файл) в корне проекта

- 📝 Функциональность этих скриптов уже встроена в `FSA-AstraInstall.py`

- Исходные файлы **НЕ изменялись** - все изменения только в выходных файлах

- Скрипты автоматически определяли пути к файлам в корне проекта
