## 📋 Описание

Эта папка содержит инструкции, руководства и техническую документацию по использованию и работе с проектом FSA-AstraInstall.

## 📁 Содержимое

### 🚀 Руководства по запуску

#### `STARTUP_ALGORITHM.md`
- **Описание:** Алгоритм гарантированного запуска FSA-AstraInstall
- **Содержит:** Описание этапов запуска, fallback механизмы, проверки зависимостей
- **Для кого:** Разработчики, системные администраторы

#### `ALGORITHM_BEHAVIOR.md`
- **Описание:** Алгоритм поведения системы установки/удаления
- **Содержит:** Принципы работы, статусы компонентов, последовательность выполнения
- **Для кого:** Разработчики

### 📦 Руководства по установке

#### `WINE_INSTALL_GUIDE.md`
- **Описание:** Руководство по установке Wine и компонентов
- **Содержит:** Инструкции по установке Wine, настройке окружения
- **Для кого:** Пользователи, системные администраторы

- **Для кого:** Разработчики, системные администраторы
