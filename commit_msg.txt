Упрощена логика установки winetricks компонентов

Изменения:
- GUI: компоненты winetricks теперь отображаются с отступами без галочек (только информация)
- Установка: выбор WINEPREFIX устанавливает ВСЕ компоненты сразу (нельзя выбрать отдельные)
- Удаление: выбор WINEPREFIX удаляет весь ~/.wine-astraregul со всеми компонентами
- Убрана логика выборочной установки/удаления отдельных компонентов
- Проверка статуса: отображается для каждого компонента отдельно

Причина изменений:
winetricks не поддерживает удаление отдельных компонентов, поэтому
нет смысла давать пользователю возможность выбирать их по отдельности.
Теперь WINEPREFIX работает как единый блок компонентов.

Преимущества:
- Проще для пользователя (1 галочка вместо 6)
- Надежнее (нет проблем с частичной установкой)
- Быстрее установка благодаря кэшу (~3-4 минуты)
- Видны статусы всех компонентов для диагностики
